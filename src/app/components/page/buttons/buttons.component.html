Buttons allow users to perform actions and choose with a single tap.

Architecture

An organized and systematic architecture of various button design elements, including size, format, appearance, variant, and state, for a cohesive and consistent design system.

Sizes

To provide clarity to users, it's crucial to implement button sizes consistently across products. Each button size serves a particular purpose, and using them appropriately impacts the user experience.

Size
Description
Regular

Use the regular button size when vertical space is limited, or when buttons are paired with input fields. It's the most commonly used button size for various use cases, including tables.

Large

Use the large button size for the most critical actions, especially those located at the top of a page. It's also suitable for buttons needed in larger components like side panels or modals.

Structure

The defined button structure comprises spacing, font and icon sizing, container dimensions, and other design elements. It's recommended to adhere to this structure for a cohesive and consistent design.

Format

Explore a variety of button formats, each offering unique visual compositions to facilitate engaging user interactions and meet diverse design needs. Select the optimal format that effectively communicates your design and enhances the overall user experience.

Variant
Description

Use the text-with-icon variant when you need to display text alongside an icon. This variant is suitable for scenarios where both the text and icon convey important information or functionality.

The text-only variant is ideal when you want to display plain text without any accompanying icons. It provides a clean and minimalistic appearance, focusing solely on the textual content.

When you need to showcase icons as standalone elements, use the icon-only variant. This variant is suitable for situations where the icon itself conveys the intended action or purpose.

The full-width variant is designed to span the entire width of its container. It is commonly used for buttons placed in horizontal navigation bars or sections that require a wider visual presence.
