The design principle of grouping refers to the organization of elements within a user interface in a way that helps users understand the relationships between them and the hierarchy of information. In a SaaS app design system, grouping can be used to create a clear and organized visual structure for the user interface.

Guidelines

Using whitespace

Placing elements in a clear and organized layout and using whitespace to separate different groups of elements can help to create a clear hierarchy and make it easier for users to process the information.

Using visual cues

Using visual cues such as color, shape, or size to group related elements can help to create a clear hierarchy and make it easier for users to understand the relationships between different elements.

Creating logical groupings

Grouping elements in a logical way based on their function or purpose can help to create a clear and organized structure for the user interface.

Avoiding clutter

Avoiding clutter by keeping the number of elements on the page to a minimum and using clear and concise language can help to create a clear and organized layout.

Examples

By applying the design principle of grouping in a SaaS app design system, you can create a clear and organized visual structure that helps users understand the relationships between different elements and the hierarchy of information. Few examples below shows how the design principle of grouping can be applied in the Chargebee app.

Navigation

Grouping navigation elements (max seven items per group) into clear categories or sections can help to create a clear hierarchy and make it easier for users to find the information they need.

Form fields

Grouping form fields into logical sections or groups can help to create a clear and organized layout and make it easier for users to complete the form.

Data visualization

Grouping data into clear and logical categories or using visual cues such as color or shape to group related data can help users understand and interpret complex data sets more easily.

Dashboards

Grouping dashboard elements into clear categories or sections can help to create a clear hierarchy and make it easier for users to understand and interact with the information being presented.
